# Meu Portfólio Web (versão: 1.0)

## Resumo

Site desenvolvido com o intuito de apresentar visualmente as minhas habilidades, experiências e onde me encontrar por meio de contatos. 

#### Objetivos para o desenvolvimento
- Responsividade: Desenvolvido para diferentes telas, desde pequenos celulares a largos computadores.
- HTML Semântico: Preocupação com o desenvolvimento do HTML de forma semântica.
- Legibilidade: Escolha de nomes intuitivos para ids e classes, além do uso de comentários para deixar o código legível mesmo a longo prazo.

### Portfólio dividido em Menu + 4 seções + página extra para confirmar contato
#### Seções
- Início
- Sobre mim
- Projetos
- Contato
